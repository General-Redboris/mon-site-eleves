## Analyser un document : la méthode pas à pas

L'analyse de document est un exercice fondamental en histoire-géographie. Voici la méthode à suivre systématiquement.

## Étape 1 : Présenter le document

Avant toute chose, tu dois **identifier** le document. Réponds à ces questions :

- **Nature** : quel type de document ? (texte, carte, photographie, graphique, affiche, caricature...)
- **Auteur** : qui a créé ce document ?
- **Date** : quand a-t-il été réalisé ?
- **Source** : d'où vient-il ? (livre, journal, archive...)
- **Sujet** : de quoi parle-t-il ?

> Astuce : retiens l'acronyme **NADS** — Nature, Auteur, Date, Sujet

## Étape 2 : Décrire le document

Décris ce que tu **vois** ou ce que tu **lis** :

- Pour un texte : relève les idées principales, les mots-clés
- Pour une image : décris les éléments visibles (premier plan, arrière-plan)
- Pour une carte : identifie le titre, la légende, l'échelle
- Pour un graphique : lis les axes, identifie les tendances

## Étape 3 : Expliquer et interpréter

C'est l'étape la plus importante. Tu dois :

1. **Replacer** le document dans son **contexte historique ou géographique**
2. **Expliquer** ce que le document nous apprend
3. **Montrer** les liens avec le cours
4. **Identifier** le point de vue de l'auteur (est-il objectif ? engagé ?)

## Étape 4 : Porter un regard critique

Pour aller plus loin :

- Ce document est-il **fiable** ? Pourquoi ?
- Que ne nous montre-t-il **pas** ?
- Quel est l'**intérêt** de ce document pour comprendre le sujet ?

## Checklist d'auto-évaluation

Avant de rendre ton travail, vérifie :

- [ ] J'ai présenté le document (nature, auteur, date, sujet)
- [ ] J'ai décrit le contenu du document
- [ ] J'ai expliqué le document en le reliant au cours
- [ ] J'ai identifié le point de vue de l'auteur
- [ ] J'ai porté un regard critique sur le document
- [ ] Mon texte est rédigé en phrases complètes
